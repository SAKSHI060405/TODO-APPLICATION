# 🌸 Pastel Daily Tasks To-Do App

A cute and aesthetic To-Do List web application built using **HTML, CSS, and JavaScript**.

This project was created to practice front-end development while experimenting with a soft pastel design theme. The interface focuses on simplicity, usability, and a visually pleasing user experience.

## ✨ Features

- ➕ Add new tasks
- ✅ Mark tasks as completed
- 🗑️ Delete tasks
- 🎀 Soft pastel pink user interface
- 📱 Responsive and beginner-friendly design
- 💖 Clean and minimal layout

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## 📸 Preview

The application includes:

- Cute pastel color palette
- Rounded card design
- Interactive task management
- Aesthetic typography and styling
